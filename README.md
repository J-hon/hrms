# A basic Human Resource Management System

The goal of this exercise is to create a simple human resource management application using GoFiber Library. 

## Requirement
The following tools are required to run the project
<ul>
<li> Go (Golang) </li> 
<li> GoFiber </li> 
<li> MongoDB</li>
</ul>
